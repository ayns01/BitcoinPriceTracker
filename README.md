# BitcoinPriceTracker

I made Bitcoin price tracker app for practice.<br>
It shows 1BTC price and diff of week, month, day and hour for some country using Bitcoin API(https://apiv2.bitcoinaverage.com/#ticker-data-per-symbol).<br>
